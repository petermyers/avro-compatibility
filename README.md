## Prerequisites

- node >=18.18.0

## Installation
```sh
npm install avro-compatibility
```
or
```sh
yarn add avro-compatibility
```
## Development
### Install
```sh
yarn install
```

### Tests
```sh
# Run tests.
yarn test

# Run tests with coverage.
yarn test:coverage

# Run lint and prettier.
yarn lint

# Fix lint errors than can be automatically resolved.
yarn format
```
